# APILearn Roadmap

This project is already usable as a static API learning playground. These additions would make it stronger as a portfolio project:

## Near-term

- Import and export request collections as JSON
- Save favorite requests in localStorage
- Add generated code snippets for `fetch`, `curl`, and Axios
- Add keyboard shortcuts for sending requests and switching tabs

## Mid-term

- Add environment variables for base URLs and tokens
- Add a small accessibility audit and ARIA improvements
- Add automated browser checks with Playwright
- Add a light theme toggle

## Long-term

- Add a mock API mode for offline lessons
- Add GraphQL request support
- Add WebSocket demo playground
